# Autonomous AI Tarot Deck Generator

## One-liner

Generate a complete, commercially viable 78-card tarot deck — stylistically cohesive
and production-ready — using a fully autonomous AI system that requires minimal human
intervention.

## Value Proposition

Creating a tarot deck from scratch is expensive and slow, and maintaining visual
consistency across 78 cards is difficult. This project defines a shared visual system,
generates every card against that system, evaluates output quality automatically, and
repairs weak generations through retry logic.

## Who It's For

- Indie tarot deck creators
- Spiritual brand founders
- Digital product entrepreneurs
- AI-native creative technologists
- Publishing studios and art directors

## Job to Be Done

When creating a complete tarot deck, the user needs a system that preserves artistic
consistency across all cards, generates production-quality images, catches issues, and
returns a finished set with minimal manual effort.

## Key Functionality

- Style Bible generation
- Per-card concept generation
- Prompt construction
- Image generation (1024×1536 PNG)
- Multimodal evaluation with structured scoring
- Prompt repair and retry loop
- Sequential orchestration over all 78 cards
- Results manifest for traceability

## Out of Scope (v1)

- UI dashboard
- Marketplace publishing flows
- Manual review gates in the core loop
- Multi-deck parallel scaling
- Custom model training/fine-tuning
